## コマンド集

```script:記事作成
 $ npx zenn new:article
```

```script:プレビュー
$ npx zenn preview # プレビュー開始
```

:::message
デフォルトでは localhost:8000 で立ち上がります。
npx zenn preview --port 3000 というようにポート番号の指定もできます。
:::
